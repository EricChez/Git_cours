CSS Flex Transitions

Nous pouvons mettre plusieurs propriétés à une transition.

Avec l'option delay, nous pouvons retarder la transition.
On peut aussi créer une classe delay qui pourra s'appliquer à plusieurs éléments.

Il existe des sites internet pour générer et personnaliser ses propres courbes de Béziers, comme par exemple :
https://cubic-bezier.com/

.longhand {
transition-property: transform;
transition-duration: 300ms;
transition-timing-function: ease-in-out;
transition-delay: 0s;
}

.shorthand {
transition: transform 300ms ease-in-out 0s;
}
