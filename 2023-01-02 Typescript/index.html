TypeScript

TypeScript est un langage de programmation développé par Microsoft en 2012, principalement parce que le code JavaScript devenait trop complexe lorsqu'il s'agissait d'applications à grande échelle.

TypeScript est un langage de programmation orienté objet. TypeScript est destiné à la construction de grandes applications et sera toujours compilé en JavaScript.

Contrairement au JavaScript dont les variables n'ont pas de type, TypeScript en a et permet ainsi une meilleure visibilité du code.

Pour activer le TypeScript, taper tsc -w main.ts dans le terminal.

inférence : TypeScript infère le type des variables que nous déclarons. On utilise uniquement let et const pour déclarer nos variables. Si un type est assigné à une variable, on ne peut plus changer le type de cette variable, celui-ci est bloqué.
