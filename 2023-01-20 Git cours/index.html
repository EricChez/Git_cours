Cours Git

Préambule

On ouvre le terminal dans le dossier du cours.

On copie le pathname du repository Git en SSH, puis on le colle dans le terminal du cours :

stagiaire15@stagiaire15-HP-ProBook-650-G4:~/Bureau/Main/2023-01-20 Git cours$ git clone git@github.com:EricChez/Git_cours.git

Pour générer un clé SSH sur windows, dans le terminal :

ssh-keygen -t rsa

Mais il y a autre chose a faire dans Github pour obtenir la clé qu'on copie avant de la coller dans le terminal.

Ouvrir un terminal gitbash dans le bureau.

Le dossier .git est caché. Pour le voir, dans le terminal du dossier :

stagiaire15@stagiaire15-HP-ProBook-650-G4:~/Bureau/Main/2023-01-20 Git cours/Git_cours$ ls -a . .. .git
stagiaire15@stagiaire15-HP-ProBook-650-G4:~/Bureau/Main/2023-01-20 Git cours/Git_cours$

My first commit

On va rester en local pour l'instant. Dans le terminal de VS Code :

git add .

Pour savoir ce qui se passe :

git status

git commit -m "first commit"
